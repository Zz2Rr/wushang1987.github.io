## Collection 接口

* 是List Set Queue接口的父接口

## List接口及其实现类 ArrayList
List 是元素有序并且可以重复的集合，被称为序列

Lsit 可以精确的控制每个元素插入的位置，或删除某个元素的位置

ArrayList数组序列，是List的一个重要实现类
